_"무엇을 했다"가 아니라 "얼마나 빨리 기여했나"를 명시 (온보딩 기간, 단독 처리 시점, 첫 배포/첫 성과까지 걸린 시간) (협업 맥락 포함: 혼자 vs 팀, 신규 vs 레거시, 마이그레이션 경험)_

- _조직 및 제품 관점에서 병목, 리스크, 비효율_

**행동(Actions)**
- _내가 직접 내린 기술적 및 구조적 판단_

**결과(Result)**
- _수치, 지표, 속도, 안정성 변화_

**영향(Impact)**
- _시간, 비용, 리스크 중 무엇이 줄었는가_
